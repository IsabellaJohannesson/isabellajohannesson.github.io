Cannie

DT2140 Multimodal Interaction and Interfaces

Visual Media

Cannie is a waste separation assistant designed using augmented reality (AR). The integrated input modality is: visuals for scanning the product, and the integrated output modalities are: AR for displaying and positioning the assistant, speech output supplying the user with information on how to sort the product, and sound output for enhancing the exaggerating movement around the objects.
